### Firebase Instructions:

-   Register App
-   Add Firebase
-   npm install firebase
-   create firebase.js in a directory within your project, we put ours in utils
-   Install Firebase CLI
-   npm install -g firebase-tools
-   build your app
-   deploy to firebase
-   firebase login
-   firebase init
-   firebase deploy
